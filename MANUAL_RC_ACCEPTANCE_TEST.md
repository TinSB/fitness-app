# IronPath Manual RC Acceptance Test

本文档用于 Release Candidate 发布前手动验收。执行前请先完成自动验证：

```bash
npm run typecheck
npm test
npm run build
```

## 八、发布判定

- P0：必须全部通过。任一 P0 失败即阻塞发布。
- P1：可带备注发布，但必须记录影响范围、截图和后续修复计划。
- P2：不阻塞发布，可进入发布后整理。

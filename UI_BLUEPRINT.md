# IronPath UI 蓝图

## 主导航

主导航固定为：今日 / 训练 / 记录 / 计划 / 我的。

- 今日：回答今天是否训练、练什么、从哪里开始。
- 训练：记录当前训练，移动端优先进入“专注训练”。
- 记录：回看训练日历、历史详情、PR / e1RM 和训练记录数据。
- 计划：管理未来训练安排、周期、训练日模板、实验模板和回滚。
- 我的：集中管理设置、筛查、单位、健康数据导入、备份和恢复。

## 中文显示规则

用户可见内容默认中文显示。训练模板示例统一写作推 A、拉 A、腿 A；底层 id 仍保持 `push-a`、`pull-a`、`legs-a`。

必要专业缩写可以保留，但必须有中文语境，例如 PR、e1RM、余力（RIR）。技术文档可以提到 Focus Mode，用户界面统一显示“专注训练”。

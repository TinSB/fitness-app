## zh-Hans（版本更新说明）

```
本次更新的重点：你的肌群发展等级来了。

• 新增「发展」板块：每块肌群一个 Lv.1–20 等级，附训练量与强度趋势——点开任何肌群，看它为什么是这个等级
• 整体训练级别（校准中 → 初级 → 中级 → 高级）与均衡度一目了然；数据不够时诚实显示「正在校准」，绝不编数字
• 新增「发展画像」分享卡：把你的肌群等级做成海报，一键分享（不含体重等隐私数据）
• 练完当天，今日页新增当日总结：总量、动作与组数、时长与 PR 徽章，顺手就能分享这场训练
• 计划页排期大幅精简：训练日构成一屏看完，点开任意训练日仍可自由增删换动作
• 修复与打磨：休息提醒更可靠、支持系统字体大小（Dynamic Type）、空状态与首屏细节全面优化
```

## en-US (What's New)

```
The headline this time: muscle development levels are here.

• New Development section: every muscle gets a Lv.1–20 rating with volume and strength trends — tap any muscle to see exactly why it's rated that way
• Overall training level (Calibrating → Beginner → Intermediate → Advanced) and a balance score at a glance; when data is thin it honestly says "Calibrating" instead of making numbers up

```

## 营销素材候选（社媒 / 商店描述补充，非 What's New 正文）

- **一句话主打**：「你的每块肌肉，都有自己的等级」/ "Every muscle earns its level."
- 差异化点（长文案可展开）：
  1. 等级不是拍脑袋——由你的真实训练量与力量趋势本地计算，数据不够时它会说「正在校准」而不是编一个数
  2. 点开每个等级都有人话依据（「近期训练量扎实」「力量突破抬升了等级起点」）——不是黑盒分数
  3. 卧推 100kg/225lb 这类公认里程碑会直接抬升相关肌群的等级起点——真实成就说话
  4. 隐私默认：等级全部本地计算；分享卡结构上不可能带出体重、伤痛、健身房位置等敏感字段

## B · 停练回归（你 7/8 抱怨的场景）

## C · 每周循环模式

- [ ] C1 设置 → 训练周期 → 有「**每周重新开始循环**」开关，默认关，下方一句说明看得懂
- [ ] C2 今日页右上「今天换一天练」→ 弹窗正常、选项清楚；开关切换前后各弹一次，描述语境跟着变
- [ ] C3 （时间依赖）开着开关跨到下周一：今日页从分化第一天开始

## D · 肌群详情页（点进去看细分）

- [ ] D1 进度页 → 点「**背部**」行 → 弹出详情页：大等级 + 进度条 + **部位构成**（背阔 / 上背 / 斜方各自的 Lv 和每周组数）+ 依据区
- [ ] D2 点「**肩部**」→ 部位构成显示前束 / 中束 / 后束
- [ ] D3 点「胸部」（无细分的肌群）→ 只有等级和依据，**没有**空的「部位构成」区块
- [ ] D4 你没单独练过的部位（比如斜方 0 组）→ 如实显示 **Lv.1 + 每周 0 组**，不虚高

## E · 相对力量标准（按体重和性别）

- [ ] E1 设置 → 训练背景 → 新增「**性别**」行（未设置显示 —）
- [ ] E2 点进去：三个选项（男 / 女 / 暂不设置）+ 说明「仅用于力量标准对比」；选「男」保存 → 铭牌行变「男」+ 出现一句修改收据
- [ ] E3 **出错场景**：再点进去换个选项但**不点保存直接取消** → 铭牌行保持原值不变
- [ ] E4 填了性别且有体重（连过 Apple 健康或档案里有）后，进度页点你的强项肌群 → 依据区**可能**出现「按体重的力量标准抬升了等级起点」（是否出现取决于你的成绩过没过档——你的卧推水平大概率能看到）
- [ ] E5 改回「暂不设置」→ 下次进进度页等级回到原状（无残留）

## F · 自动均衡（弱肌群自动多安排）

> 前置：进度页「发展」里得先有带「**正在补足**」标签的肌群（数据少全在校准中就不会触发——正常，练几场再看）。

- [ ] F1 有「正在补足」肌群时，正常训练日：该肌群为主的动作**组数比平时多 1 组**（比如弯举 2 组变 3 组）
- [ ] F2 点「查看依据」展开 → 最底部一行「**XX正在补足　今天多安排了组数**」
- [ ] F3 页面其他任何地方**没有**新增说明文字（无小字——你拍板的红线）
- [ ] F4 轻练日 / 减载日：组数回到正常值（自动让位，不加）
